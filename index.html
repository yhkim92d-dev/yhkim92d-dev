Line Tracer

1. Project Name: Delivery Robot Challenge (Line Tracer Project)
2. Project Description: Participated in the "2023 Gumi Electronics Technical High School Hackathon Competition," implementing a delivery robot to participate in the food delivery contest using a line tracing robot.
3. Duration: November 18, 2023 - November 19, 2023
4. Key Responsibilities: Firmware development.
5. Progress:
- Controlled DC and Servo motors for precise movement.
- Developed and programmed autonomous control using gyro sensors to adjust angles.
- Implemented line tracing functionality through line sensor interfacing.
- Designed navigation functionality using a Huskylens camera for tracking.

Digital Clock

1. Project Name: Digital Clock
2. Project Description: Implementation of specific features of a Digital Clock using ATmega128.
3. Duration: April 2024 - June 2024
4. Key Responsibilities: Firmware development and project planning.
5. Progress:
- Analyzed necessary components for implementing the Digital Clock and designed the circuit diagram using OrCAD.
- After completing the circuit diagram, assembled the hardware according to the design.
- Developed algorithms for selected features of the Digital Clock and wrote the corresponding code.
- Performed step-by-step validation during code development.
- After completing the code, verified alignment with the project plan and debugged any errors encountered.
